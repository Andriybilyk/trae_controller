# Схема підключення контролера печі (Trae Controller)

Цей документ описує повну схему підключення всіх компонентів, включаючи силову частину, живлення та периферію.

## ⚡ Архітектура живлення

Для надійної роботи системи рекомендується використовувати окремий блок живлення 12V (або 24V залежно від вентилятора) та понижуючий перетворювач (Buck Converter) для живлення електроніки.

```mermaid
graph TD
    Mains[Мережа 220V] --> PSU[Блок живлення 12V DC]
    Mains --> SSRs[Силові реле (SSR)]
    
    PSU -->|12V| Fan[Вентилятор охолодження]
    PSU -->|12V| Buck[DC-DC Buck Converter (LM2596/XL4015)]
    
    Buck -->|5V| ESP32[ESP32-S3 DevKitC (Pin 5V/VIN)]
    Buck -->|5V| Relays_Logic[Живлення логіки реле (VCC)]
    
    ESP32 -->|3.3V| Display[Дисплей ILI9341]
    ESP32 -->|3.3V| Max31855[Термопара MAX31855]
    
    subgraph "Керування вентилятором"
    GPIO13[GPIO 13] -->|PWM/Logic| Gate[Затвор MOSFET]
    Gate --> MOSFET[N-Channel MOSFET (IRLZ44N)]
    MOSFET --> Fan
    end
```

## 📋 Список компонентів

1.  **ESP32-S3 DevKitC-1**: Основний контролер.
2.  **Блок живлення (PSU)**: 12V 2A (мінімум).
3.  **DC-DC Buck Converter**: Наприклад, LM2596. Вхід 12V -> Вихід 5V. Налаштуйте вихід на 5.0-5.1V перед підключенням до ESP32!
4.  **Вентилятор**: 12V комп'ютерний вентилятор (80мм або 120мм) для охолодження SSR та електроніки.
5.  **MOSFET**: N-Channel Logic Level MOSFET (наприклад, **IRLZ44N** або **AO3400**). Звичайний IRF540 може не відкритися повністю від 3.3V.
6.  **Резистори**:
    *   **220 Ом**: Між GPIO 13 та затвором (Gate) MOSFET (обмеження струму).
    *   **10 кОм**: Між затвором (Gate) MOSFET та землею (GND) (підтяжка до землі, щоб закрити транзистор при запуску).
7.  **Дисплей**: ILI9341 SPI (2.8" або 3.2").
8.  **Термопара**: K-Type з модулем MAX31855 (3.3V версія).
9.  **SSR (Твердотільні реле)**: 3 шт. (наприклад, SSR-40DA). Встановлювати на радіатор з термопастою!

## 🔌 Детальна карта підключень (Pinout)

### 1. Живлення та Земля
*   **12V PSU (+)** -> Вхід Buck Converter (+), Вентилятор (+)
*   **12V PSU (-)** -> Вхід Buck Converter (-), Витік (Source) MOSFET, Спільна земля (GND) всіх модулів
*   **Buck Converter Out (+5V)** -> ESP32 5V (або VIN), VCC модулів реле
*   **Buck Converter Out (GND)** -> ESP32 GND

### 2. Вентилятор (через MOSFET)
*   **Вентилятор (+)** -> 12V PSU (+)
*   **Вентилятор (-)** -> MOSFET Drain (Сток, середня ніжка)
*   **MOSFET Source (Витік, права ніжка)** -> GND
*   **MOSFET Gate (Затвор, ліва ніжка)** -> Резистор 220 Ом -> **GPIO 13**
*   **Резистор 10 кОм**: Один кінець до Gate, інший до GND.
