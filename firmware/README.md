# Trae Controller Firmware

This project contains the firmware for the Trae Controller, an ESP32-S3 based device.

## Features

*   Web interface for configuration and monitoring.
*   Thermal control with PID controller.
*   WiFi connectivity.

## Building the project

1.  Install [PlatformIO](https://platformio.org/).
2.  Clone this repository.
3.  Open the project in Visual Studio Code with the PlatformIO extension.
4.  Build and upload the project to the device.
